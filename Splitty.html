Project Overview

Splitty is a money splitting app for simplifying paying back the money you owe. It was created as an assignment for the TU delft with a team of six. It was created in java using spring boot.

My focus for this project

I mostly worked on the API for this project both client and server side. For this we had to both use websockets and long polling.
